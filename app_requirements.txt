
【年中向けタッチ操作ブラウザ学習アプリ：要件定義】

■ 対象
- 年中（4歳・5歳）向け
- タブレット・スマホ・PCのブラウザ対応（タッチ操作前提）

■ 目的
- 文字や数字に対する興味を引き出す
- 楽しみながら学習する習慣づけ
- 問題をクリアして達成感を得られる構成

■ 学習構成
- 1回の学習で3つの問題がランダム出題される
- 1問あたり約5分、全体で約15分の学習体験
- 出題ジャンル例：
  - ひらがな（文字と言葉のマッチ）
  - 数字（かぞえる、数字と物の一致）
  - 図形・パターン（同じ形探し、法則）

■ 基本構成
1. トップ画面（あいさつと「スタート」ボタン）
2. ランダム問題（3問連続で出題）
3. ごほうび画面（クリア報酬、メダルやシール演出）
4. リトライ or 終了ボタン

■ 機能要件
- 画面UIは大きく、誤タップを防止
- 全出題・選択肢は画像＋音声対応（非読者前提）
- 音声読み上げ（Web Speech API or 音声ファイル）
- タッチイベント対応（モバイル・タブレット最適化）

■ 技術要件
- HTML / CSS / JavaScript (VanillaJS または React可)
- GitHub Pagesで静的ホスティング対応
- assets/images/ に画像素材、assets/sounds/ に音声素材
- 出題データは JSONで管理（ランダム選出用）
- 外部サーバー・データベースは使用しない（完全クライアント）

■ 公開方法
- GitHubリポジトリに配置し、GitHub Pagesでホスト
- URLを保護者や教育者に共有し、どこでもプレイ可能
